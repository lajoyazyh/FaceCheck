可以，按这个顺序在 Windows 上启动最稳。

  当前 App 真正可实测的是：

  - 登录
  - 首页
  - 个人资料页
  - 个人人脸照片页

  1. 先启动基础设施

    在 PowerShell 里：

  cd C:\Users\alvinding\desktop\FaceCheck
  docker compose up -d postgres redis rabbitmq
  docker compose ps

  端口会是：

  - PostgreSQL: 5432
  - Redis: 6379
  - RabbitMQ AMQP: 5672
  - RabbitMQ 管理台: http://localhost:15672

  RabbitMQ 本地账号按当前 docker-compose.yml 是：

  - 用户名：facecheck
  - 密码：facecheck

  2. 启动后端

    先说明一件事：Flutter 不需要你改系统默认 JDK。当前 Android 构建已经验证过，直接走
    Android Studio 自带 JBR。
    后端如果你当前默认 Java 不是 21，最多只在这个 PowerShell 窗口里单独切 JAVA_HOME，不要改
    全局默认 JDK。

  在新的 PowerShell 窗口里：

  cd C:\Users\alvinding\desktop\FaceCheck\backend

  $env:SPRING_PROFILES_ACTIVE="local"
  $env:DB_URL="jdbc:postgresql://localhost:5432/facecheck"
  $env:DB_USERNAME="facecheck"
  $env:DB_PASSWORD="facecheck"

  $env:REDIS_HOST="localhost"
  $env:REDIS_PORT="6379"

  $env:RABBITMQ_HOST="localhost"
  $env:RABBITMQ_PORT="5672"
  $env:RABBITMQ_USERNAME="facecheck"
  $env:RABBITMQ_PASSWORD="facecheck"

  $env:JWT_SECRET="change-this-to-a-long-random-local-secret"
  $env:HUAWEI_CLOUD_ENABLED="false"

  $env:SPRING_RABBITMQ_LISTENER_SIMPLE_AUTO_STARTUP="true"

 .\mvnw.cmd spring-boot​​

  说明：

  - HUAWEI_CLOUD_ENABLED=false 时，后端会走 mock FRS，OBS 也是本地 mock 存储实现，不需要真
    实华为云凭证。
  - SPRING_RABBITMQ_LISTENER_SIMPLE_AUTO_STARTUP=true 很重要。
    当前 backend/src/main/resources/application-local.yml:1 默认把 listener 关了；如果不显
    式开，人脸照片上传后会一直停在 PENDING，不会异步注册完成。

  后端起来后先验证：

  - http://localhost:8080/api/health

  4. 启动 Android App

    建议先用 Android 模拟器，不要先上真机。

  原因是当前 app/lib/shared/config/app_env.dart:1 对 Android 固定用了：

  - http://10.0.2.2:8080

  这只适用于 Android Emulator。

  先在 Android Studio 里启动模拟器，然后在 PowerShell 里：

  cd C:\Users\alvinding\desktop\FaceCheck\app
  C:\Users\alvinding\flutter\bin\flutter.bat pub get
  C:\Users\alvinding\flutter\bin\flutter.bat devices
  C:\Users\alvinding\flutter\bin\flutter.bat run -d <你的模拟器deviceId>

  如果只开了一个模拟器，通常直接：

  C:\Users\alvinding\flutter\bin\flutter.bat run

  就行。

  6. 如果你想用真机而不是模拟器

    当前代码不能直接用真机连后端，因为 10.0.2.2 是模拟器专用地址。
    真机要先手改 app/lib/shared/config/app_env.dart:1：

  把 Android 分支的 baseUrl 改成你 Windows 主机的局域网 IP，比如：

  baseUrl: 'http://192.168.1.23:8080'


  - 后端起来了，但照片一直不转 ACTIVE
      - 你大概率没加 SPRING_RABBITMQ_LISTENER_SIMPLE_AUTO_STARTUP=true
  - App 登录报网络错
      - 先确认后端健康检查通了
      - 模拟器场景必须是 10.0.2.2:8080
  - 用真机连不上
      - 这是当前 app_env.dart 的限制，不是后端坏了
